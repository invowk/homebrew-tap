# Homebrew Tap for invowk

This repository contains the [Homebrew](https://brew.sh) formula for [invowk](https://github.com/invowk/invowk) â€” a dynamically extensible command runner.

## Installation

```bash
brew install invowk/tap/invowk
```

## Upgrading

```bash
brew upgrade invowk
```

## Formula Updates

The formula in this tap is automatically updated by [GoReleaser](https://goreleaser.com) on each stable release of invowk.
